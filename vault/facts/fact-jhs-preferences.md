# 장희성의 취향

- **풋살을 좋아한다** — 주요 취미이자 운동.
- **한식을 좋아한다** — 음식 취향의 중심.
- **대중문화에 관심이 없다** — 연예, 드라마, 유행 콘텐츠 등은 관심 영역 밖.

활용 지침: 강의 예시나 비유를 만들 때 대중문화 레퍼런스(연예인, 드라마 등)는 피하고, 풋살·축구나 음식 같은 소재가 본인에게 더 와닿는다.
